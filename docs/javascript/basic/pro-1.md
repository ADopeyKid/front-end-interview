# 什么是原型，原型有什么特点?

?> `原型`是一个对象，其他对象可以通过它实现属性继承, 只有null除外，它没有自己的原型对象。

> 原型对象上的所有属性和方法，都能被对应的构造函数创建的实例对象共享（这就是 JavaScript 继承机制的基本设计），也就是说，不必在构造函数中定义对象实例的信息，而是可以将这些信息直接添加到原型对象中

> 每一个构造函数都有一个prototype（原型）属性，这个属性就是使用构造函数创建出来的实例对象的原型对象
