# 数组原型是数组

```js
Array.isArray( Array.prototype )
```

?> `结果是` true

> **Array.prototype.constructor**

> 所有的数组实例都继承了这个属性，它的值就是 Array，表明了所有的数组都是由 Array 构造出来的。
