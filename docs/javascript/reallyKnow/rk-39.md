# min与max共舞

```js
var min = Math.min(), max = Math.max()
console.log(min < max);
```

?> `结果是` false

?> `Math.min`描述：如果没有参数，结果为Infinity。

?> `Math.max`描述: 如果没有参数，结果为-Infinity。
