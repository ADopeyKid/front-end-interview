# 最小的正值

```js
Number.MIN_VALUE > 0
```

?> `结果是` true

> `Number.MIN_VALUE` 属性表示在 JavaScript 中所能表示的最小的正值

> `MIN_VALUE` 属性是 JavaScript 里最接近 0 的正值，而不是最小的负值

> `MIN_VALUE` 的值约为 `5e-324`。小于 `MIN_VALUE` ("underflow values") 的值将会转换为 `0`。
