# 如何合并数组中相邻且重复的元素?

```js
// 示例
merge([3,2,2,4,5,5,6,2,1]);  //  输出[3,2,4,5,6,2,1] 
merge([3,2,3]);  //  输出[3,2,3] 
merge([2,2,3]);  //  输出[2,3] 
```

**答案：**

```
(arr + ',').replace(/(\d+,)\1+/ig, '$1').split(',').slice(0, -1);  
```
